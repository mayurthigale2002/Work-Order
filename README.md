Work-Order

A full-stack Work Order Management System built using modern web technologies.
This project helps manage work orders, bills, proposals, and related operations through a structured UI and backend API.

📌 Overview

This application allows users to:

Create and manage work orders
Generate and track bills
Handle proposals
View and update records efficiently

It is designed for internal business workflow automation.

🚀 Tech Stack
Frontend
React.js
JavaScript (ES6+)
Axios
React Router
Backend
Node.js
Express.js
REST API
Database
Mysql (used in backend)

⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/mayurthigale2002/Work-Order.git
cd Work-Order

3. Setup Backend
cd backend
npm install
npm start

4. Setup Frontend
cd frontend
npm install
npm run dev   # or npm start depending on setup
